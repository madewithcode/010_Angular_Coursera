Create and Use Promise

CREATE promise

  1. create service A (with .service)
  2. inject $q into the service
  3. create a variable for 'this' to provide a handle on the service methods
  4. create a method on the service
    • ---- start with deferred = $q.defer()
    • ---- execute method logic
    • ---- once executed, complete by either deferred.resolve() or deferred.reject()
    • ---- end with returning deferred.promise
  5. > each method that needs to create a promise, should have its own set of above

USE the promise in service B

  1. inject service A in service B as you want to use a method of service A in service B
  2. get a handle on the promise
    • ---- promise = A.methodYouWantToCall(args)
    • ---- implement two functions on the promise,
    • ---- way1: promise.then( function(success) {} ).catch( function(error) {} )
    • ---- way2: promise.then( function(success) {}, function(error) {} )
  3. way1 is more suited for running multiple promises in parallel
    • ---- promiseOne = A.methodYouWantToCall(args)
    • ---- promiseTwo = A.anotherMethodYouWantToCall(args)
    • ---- $q.all(promiseOne, promiseTwo).then( function(success) {} ).catch( function(error) {} )
